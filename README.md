# -Sistem-Deteksi-Anomali-pada-Data-Sensor-IoT-NASA-Bearing

Proyek ini bertujuan untuk membangun sistem deteksi anomali pada data sensor IoT menggunakan model hybrid CNNâ€“LSTM. Data yang digunakan berupa sinyal getaran bearing dari NASA Bearing Dataset (2nd test) yang direkam menggunakan sensor accelerometer dalam konteks Industrial Internet of Things (IIoT).

Model CNN digunakan untuk mengekstraksi fitur dari sinyal getaran pada setiap window, sedangkan LSTM digunakan untuk mempelajari pola perubahan sinyal secara temporal. Model dilatih untuk melakukan klasifikasi biner, yaitu kondisi normal dan anomali.

Proyek ini dibuat sebagai tugas pembelajaran untuk memahami penerapan deep learning pada data deret waktu sensor IoT.
